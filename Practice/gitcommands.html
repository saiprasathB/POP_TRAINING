GIT & GITHUB :

  1. Git is a version control system that helps track changes in code.
  2. GitHub is a cloud-based platform that lets you store, share, and collaborate on Git repositories online.

Basic Git Commands

  1. Configuration Commands

    • git config --global user.name "Your Name" # Set username
    • git config --global user.email "your-email@example.com" # Set email
    • git config --list # View configuration settings

  2. Repository Management

    • git init # Initialize a new Git repository
    • git clone (repository_url) # Clone an existing repository

  3. Working with Changes

    • git status # Check the status of changes
    • git add (file) # Stage a specific file
    • git add . # Stage all changes
    • git commit -m "Your commit message" # Commit changes
    • git log # View commit history

  4. Branching & Merging

    • git branch # List branches
    • git branch # Create a new branch
    • git checkout # Switch to a branch
    • git merge # Merge a branch into the current branch

  5. Remote Repositories (GitHub)

    • git remote add origin # Link local repo to GitHub
    • git push -u origin main # Push local repo to GitHub
    • git pull origin main # Pull latest changes from GitHub

  6. Undo & Reset

    • git reset --soft HEAD~1 # Undo last commit (keep changes)
    • git reset --hard HEAD~1 # Undo last commit (delete changes)
    • git revert # Revert a specific commit
